พื้นที่ศึกษา
จังหวัดอำนาจเจริญ เพราะ มีความเหมาะสมสำหรับการศึกษาด้านภูมิศาสตร์และภัยแล้ง เนื่องจากตั้งอยู่ในภาคตะวันออกเฉียงเหนือที่มีภูมิอากาศแบบมรสุม ทำให้มีฤดูฝนและฤดูแล้งแตกต่างกันชัดเจน ส่งผลให้สภาพพืชพรรณและความชื้นเปลี่ยนแปลงเด่นชัดตามฤดูกาล นอกจากนี้ พื้นที่ส่วนใหญ่เป็นดินทรายที่อุ้มน้ำน้อย และมีการใช้ที่ดินเพื่อการเกษตรที่พึ่งพาน้ำฝน จึงมีความเสี่ยงต่อภัยแล้งสูง อีกทั้งภูมิประเทศค่อนข้างราบ ทำให้ข้อมูลจากดาวเทียมมีความต่อเนื่องและเหมาะสำหรับการวิเคราะห์ดัชนีในการประเมินความเสี่ยงภัยแล้งได้อย่างมีประสิทธิภาพ
คำถามวิจัย
•	พื้นที่ใดมีความหนาแน่นของพืชพรรณสูง/ต่ำ และมีการเปลี่ยนแปลงระหว่างฤดูหรือไม่ 
•	พื้นที่ใดมีความชื้นสะสมหรือมีแหล่งน้ำผิวดิน 
•	พื้นที่ใดมีความเสี่ยงต่อภาวะแห้งแล้ง
